1. Descripción del sistema
Este sistema simula el proceso de evaluación de contenidos en una plataforma de streaming. El usuario puede ingresar diferentes tipos de contenido como películas, series, documentales o eventos. Junto con datos como duración, clasificación, horario y nivel de producción.
El programa analiza esta información y toma una decisión automática: publicar, enviar a revisión o rechazar, además de llevar un control de estadísticas durante la ejecución.

2. Instrucciones para ejecutarlo
Para ejecutar el programa, se debe abrir en un entorno compatible con C# como Visual Studio y correr el archivo principal.
Al iniciar, se mostrará un menú con varias opciones. El usuario debe ingresar el número de la opción deseada y seguir las instrucciones en pantalla.
Para evaluar contenido, se deben ingresar los datos solicitados correctamente. El sistema procesará la información y mostrará el resultado en consola.

3. Explicación del proyecto
El proyecto está desarrollado en C# utilizando estructuras fundamentales de programación como switch para el menú, ciclos do-while para mantener el sistema en ejecución, y múltiples condiciones if/else para validar reglas.
Primero se realiza una validación técnica para verificar si el contenido cumple con los requisitos establecidos. Luego, se clasifica el impacto en bajo, medio o alto. Finalmente, el sistema toma una decisión basada en estos resultados y actualiza las estadísticas generales.
