1. Descripción del sistema

Este programa es como un ayudante que revisa videos antes de mostrarlos. El usuario escribe qué tipo de contenido es, cuánto dura, a qué hora se quiere mostrar y otras características. El programa revisa esa información y decide si el contenido se puede mostrar, si necesita revisión o si no se puede aceptar.

2. Instrucciones para ejecutarlo

Para usar el programa, primero se abre en la computadora y se ejecuta. Luego aparece un menú con opciones. El usuario debe escribir un número para elegir qué quiere hacer. Si quiere evaluar contenido, debe ingresar los datos que el programa le pide. Después el programa muestra el resultado en la pantalla.

3. Explicación del proyecto

El programa funciona siguiendo pasos. Primero revisa si el contenido cumple las reglas. Luego analiza qué tan importante es. Finalmente toma una decisión. Todo esto lo hace usando instrucciones que le dicen qué hacer en cada caso y repitiendo el menú para que el usuario pueda seguir usándolo varias veces.
